# HR Analytics Project: Employee Performance Drivers

## Overview
This project analyzes employee performance drivers using Excel, Python, and Power BI.

## Objective
The goal of this project is to identify which factors most strongly influence employee performance and to present the results through statistical analysis and dashboard visualization.

## Tools Used
- Excel
- Python
- Power BI

## Project Workflow
1. Cleaned and prepared the dataset in Excel
2. Conducted analysis and regression modeling in Python
3. Built an interactive dashboard in Power BI
4. Summarized insights in a professional report

## Key Insights
- Ambition showed the strongest positive relationship with employee performance
- Emotional stability also had a positive impact
- Experience had limited practical effect
- The dashboard helps visualize performance patterns and workforce insights

## Project Files
- `data/` contains the cleaned dataset
- `excel-analysis/` contains Excel-based analysis
- `python-analysis/` contains the Jupyter notebook
- `powerbi-dashboard/` contains the Power BI dashboard
- `outputs/` contains screenshots and report exports
