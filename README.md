# 📊 HR Analytics Dashboard: Employee Performance Drivers

## 🔍 Project Overview

This project analyzes key factors that influence employee performance using a combination of **Excel, Python, and Power BI**.

The goal was to identify patterns between attributes such as **ambition, emotional stability, and experience** and their impact on overall performance.

---

## 🧰 Tools & Technologies

* **Excel** → Data cleaning, regression analysis
* **Python (Pandas, Matplotlib/Seaborn)** → Data exploration & visualization
* **Power BI** → Interactive dashboard & insights

---

## 📁 Project Structure

* `data/` → Cleaned dataset
* `excel-analysis/` → Excel outputs & regression visuals
* `python-analysis/` → Python analysis & plots
* `powerbi-dashboard/` → Final dashboard (.pbix + PDF)

---

## 📊 Key Insights

* Employees with higher **ambition** tend to show slightly higher performance
* **Emotional stability** shows a moderate positive relationship with performance
* Experience contributes, but not as strongly as behavioral traits
* High performers cluster around higher ambition + stability ranges

---

## 📈 Dashboard Highlights

* KPI metrics (Average Performance, High Performers, Total Employees)
* Scatter plots showing relationships between variables
* Interactive filters by experience, gender, and ethnicity
